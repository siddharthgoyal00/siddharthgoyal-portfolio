## Portfolio (React + TypeScript + Tailwind)

Development

```bash
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Personalize content in:
- `src/App.tsx` (name, photo, links, email, sections)
- `src/components/Header.tsx` (social links)

Font: Roboto Mono is loaded via Google Fonts in `index.html`. Layout is centered with large side padding and projects display as two per row on desktop.
